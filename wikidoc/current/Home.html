Titan is a distributed graph database optimized for processing massive-scale graphs represented over a cluster of machines and can scale elastically as your data or user base grow. Titan is a native Blueprints enabled graph database and as such, it supports the full TinkerPop stack of technologies. Titan has a pluggable storage architecture which allows it to build on proven database technology such as Apache Cassandra, Apache HBase, or Oracle BerkeleyDB and makes integration into existing database environments seamless.






Please use Aurelius’s mailing list for all Titan related questions. For TinkerPop related discussions, use the Gremlin-Users mailing list.
For commercial Titan support, please contact the graph experts at Aurelius.
