TITAN

A Highly Scalable, Distributed Graph Database

Download this project as a .zip file

Titan is a highly scalable graph database optimized for storing and querying large graphs with billions of vertices and edges distributed across a multi-machine cluster. Titan is a transactional database that can support thousands of concurrent users.

In addition, Titan provides the following features:

  • Support for ACID and eventual consistency models
  • Support for various storage layers which simplifies integration into existing database environments
    • Apache Cassandra
    • Apache HBase
    • Oracle BerkeleyDB
  • Data distribution and replication for performance and safety
  • Native implementation of the Blueprints graph API and support for the full TinkerPop stack of technologies:
    • Gremlin graph traversal language
    • Frames graph-to-object mapper
    • Rexster graph server
  • Open source with the liberal Apache 2 license

Download Titan 0.1 or clone from github.  Join the Titan mailing list to stay up-to-date.

Quick Start 

<dependency>
   <groupId>com.thinkaurelius</groupId>
   <artifactId>titan</artifactId>
   <version>0.1.0</version>
</dependency>


Graph graph = TitanFactory.open("/tmp/local-titan")
Vertex a = graph.addVertex();
Vertex b = graph.addVertex();
a.setProperty("name", "hercules");
b.setProperty("name", "cerberus");
Edge e = graph.addEdge(a, b, "battled");
e.setProperty("time", 12);

Check out the Titan documentation for more information.
